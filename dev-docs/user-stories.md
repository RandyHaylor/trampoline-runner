# User Stories

- As a player, I see my character automatically running/moving forward so I can focus on timing jumps.
- As a player, I bounce upward when I land on a trampoline so the game feels dynamic.
- As a player, I bounce higher when I land near the center of a trampoline and lower near the edges so positioning matters.
- As a player, I collect coins as I pass through them to increase my score.
- As a player, I lose or take damage when I collide with a floating enemy so I must dodge them.
- As a player, I play the game in a mobile browser so I can play anywhere without installing an app.
